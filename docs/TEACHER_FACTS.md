## 行銷標題對應關係

| 標籤 | 對應師資 |
|---|---|
| 臺（國立臺灣大學） | Barney（資工所碩士）、Seba（材料系學士） |
| 交（國立交通大學） | Barney（資工系學士） |
| 高師大（國立高雄師範大學） | Joker（英語學系畢業） |
| 嘉大（國立嘉義大學） | 豪理（應用物理學系學士） |
| 市北大（臺北市立大學） | Jason（數學系學士） |

> 縮寫版（標題、icon 區）：`臺、交、高師大、嘉大、市北大`
> 全名版（描述、meta、SEO）：`臺、交、高師大、嘉義大學、臺北市立大學`
> **絕對不可再出現「清」**：本班沒有清大師資；舊資料中的「清大物理」純屬筆誤。

未來增刪老師時：
- 如果新老師的學校已在此 5 大之內，文案不必動。
- 如果新老師來自不同學校，請評估是否要加進標題清單；若 5 個位置已飽和，請選擇將最不具差異化的標籤替換掉。
- 若有老師離職，請檢查站上是否有依賴該老師學歷的標籤；若該標籤只對應一位老師且該老師離職，需從文案中移除該標籤。
