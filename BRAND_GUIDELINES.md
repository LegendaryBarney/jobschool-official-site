## 1. 品牌核心

### 1.1 品牌識別

| 項目 | 內容 |
|---|---|
| **正式名稱** | 賈伯斯數理教室 |
| **英文/拉丁化** | Jobs Math & Science Academy（縮寫 Jobs Academy） |
| **品牌類型** | 地方性精英小班補習機構 |
| **創立年份** | 13 年以上教學資歷（可標示為 EST. ◯◯◯◯，請補上實際開業年份） |
| **總部** | 嘉義市東區康樂街 10 號 |

### 1.2 品牌定位三句話（任何外包 Brief 必附）

1. 我們是嘉義在地的精英小班補習機構，專攻國中至高中的數理科目。
2. 由台大資工碩士領軍，師資皆具台清交背景與 13 年以上教學經驗。
3. 我們相信「個別化關注」勝過大班灌輸，所以每位學生都能被看見。

### 1.3 品牌個性（人格化）

如果賈伯斯數理教室是一個人，他是：

- **35 歲的資深教師**，台大畢業後選擇回家鄉嘉義開補習班
- 個性 **沉穩、專業，但不嚴肅**；像一位你信得過的咖啡館老闆
- 說話 **精準、有條理**，會把複雜觀念講得簡單
- 對教學 **有熱情但不浮誇**，不會說「保證上台大」這類話
- 衣著乾淨整潔，常穿米色針織衫與深色長褲，戴細框眼鏡

### 1.4 Tone of Voice 語調規範

| 我們是 | 我們不是 |
|---|---|
| 溫潤、有溫度（像咖啡館） | 冰冷、企業化 |
| 專業、有依據（引用數據與經驗） | 浮誇、灌雞湯 |
| 平易近人（用學生聽得懂的話） | 學究、咬文嚼字 |
| 真誠（會承認不擅長的事） | 全能、無敵 |
| 有耐心（願意解釋細節） | 快速、激進、催促 |

**寫文案時請避免的詞彙**：
- ❌「保證上 X 大」「100% 進步」「絕對」
- ❌「業界第一」「最強」「最厲害」
- ❌「秒懂」「開掛」「無腦」（與品牌調性不符）
- ❌「親愛的家長/同學」（過時）

**推薦詞彙**：
- ✅「適合」「協助」「陪伴」「累積」「踏實」「穩」「個別化」「量身規劃」
- ✅「我們發現」「在我們的經驗裡」「過去 13 年來」

## 2. Logo 規範

> ⚠️ **目前狀態**：Logo 尚未正式設計／重整。本段為「未來 Logo 重做時應遵守的規範」與「過渡期使用建議」。
> 建議第一個月內委外重整 Logo，預算區間 15,000-40,000 元。

### 2.1 Logo 設計方向（給設計師的 Brief）

- **必含元素**：「賈伯斯」中文 + 咖啡杯象徵（呼應品牌名與咖啡館調性）
- **可考慮元素**：粉筆字感、書本、簡化的數學符號
- **避免元素**：過於卡通、過於企業化的盾牌/獎章、彩虹色
- **風格關鍵字**：`minimalist wordmark`, `cafe-inspired`, `chalk texture accent`, `serif logotype`
- **需交付**：SVG / PNG（透明底）/ AI 原始檔；含 6 種尺寸版本（favicon 16px → 大尺寸 1024px）

### 2.2 Logo 使用規則（重做後請遵守）

| 規則 | 說明 |
|---|---|
| **安全空間** | Logo 周圍至少預留「咖啡杯高度」的留白 |
| **最小尺寸** | 數位 24px、印刷 8mm |
| **彩色版本** | 主色（Espresso Tan #8B6F47）/ 全黑 / 全白 三版本 |
| **背景使用** | 淺色背景用主色 Logo；深色背景用白色 Logo |
| **禁止** | 拉伸變形、加陰影特效、改變比例、置於低對比背景 |

### 2.3 過渡期使用建議

在新 Logo 完成前：
- 統一使用「賈伯斯數理教室」純文字 wordmark（思源宋體 Noto Serif TC Bold）
- 字色用 Charcoal Bean `#1F1B16`，背景用 Latte Foam `#F5EFE6`
- 暫不使用任何視覺符號，避免新舊視覺混淆

## 3. 配色系統

### 3.1 主要色票

| 用途 | 名稱 | HEX | RGB | 使用情境 |
|---|---|---|---|---|
| **主背景** | Latte Foam（拿鐵奶泡） | `#F5EFE6` | 245, 239, 230 | 全站主背景、貼文背景 |
| **次背景** | Cream Beige（奶油米） | `#EAE0D0` | 234, 224, 208 | 區塊區隔、卡片背景 |
| **品牌主色** | Espresso Tan（濃縮咖啡棕） | `#8B6F47` | 139, 111, 71 | Logo、強調文字、次要按鈕 |
| **主文字色** | Charcoal Bean（炭咖啡） | `#1F1B16` | 31, 27, 22 | 內文、標題、深色元素 |
| **純白** | Chalk White（粉筆白） | `#FAFAF7` | 250, 250, 247 | 卡片、彈窗、對比區塊 |
| **CTA 點綴** | Caramel Gold（焦糖金） | `#C8A165` | 200, 161, 101 | 主要按鈕、超連結、強調點 |
| **警示色** | Burnt Sienna（焦土赭） | `#A14A3A` | 161, 74, 58 | 錯誤訊息、限時優惠標籤 |

### 3.2 使用比例（黃金比例 60-30-10）

- **60%**：Latte Foam（主背景）
- **30%**：Charcoal Bean + Cream Beige（文字 + 區塊）
- **10%**：Caramel Gold + Espresso Tan（強調 + CTA）
- **< 1%**：Burnt Sienna（僅警示）

### 3.3 配色禁忌

- ❌ 不使用任何純飽和色（如純紅 #FF0000、純藍 #0000FF）
- ❌ 不使用螢光色、霓虹色、漸層彩虹
- ❌ 不使用紫色系、粉紅色系（與咖啡館調性衝突）
- ❌ 整張圖卡的色票超過 5 色

## 5. 視覺元素與風格

### 5.1 核心風格關鍵字（給 AI 工具如 Midjourney / Figma AI / v0 用）

```
cafe-inspired aesthetic, warm beige tones, minimalist warmth,
chalk-line illustrations, editorial layout, generous whitespace,
hand-drawn underlines, asymmetric grid, soft grain overlay,
fraunces serif typography, subtle parallax, scroll-triggered reveal,
analog meets digital, cozy academic, matte finish
```

### 5.2 插畫風格

- **線條**：細線（1-1.5pt），手繪粉筆感，避免幾何精確感
- **色彩**：單色或雙色，多用 Espresso Tan 或 Charcoal Bean
- **題材**：咖啡杯、書本、粉筆、簡化數學符號（π, ∑, ∫）、教室物件
- **避免**：3D 渲染、卡通人物、寫實風、漫畫風

### 5.3 攝影風格

| 項目 | 規範 |
|---|---|
| **色調** | 暖色調（色溫 4500-5500K），略偏黃 |
| **光線** | 自然光優先，避免閃光燈直打 |
| **構圖** | 留白多、不擁擠，可帶咖啡杯/書本道具 |
| **後製** | 統一 Lightroom 預設或 VSCO `A6` / `KK1` 風格 |
| **避免** | HDR 過曝、磨皮過頭、冷色偏藍、手機美顏濾鏡 |

**必拍題材清單**：
- 教室空間（廣角、有溫度）
- 老師上課剪影（不正面、強調氛圍）
- 學生筆記特寫（手寫感）
- 咖啡 + 教材的桌面
- 黑板與粉筆字

### 5.4 紋理與圖案

- **可用**：紙張紋理（subtle paper texture）、輕微顆粒（grain overlay）、細格線（chalk grid）
- **禁用**：木紋、皮革、金屬、霓虹發光、玻璃反射

## 6. 動效與互動規範（網站專用）

### 6.1 全站動效原則

- **每頁主特效不超過 2 個**，避免「家具行燈光秀」感
- **動效時長**：300-600ms 為主，最長不超過 1.2s
- **緩動曲線**：預設 `cubic-bezier(0.4, 0, 0.2, 1)`（Material Design ease）
- **避免**：閃爍、彈跳過大（bounce > 1.2 倍）、無限循環的吸睛動畫

### 6.2 場景對應動效

| 場景 | 動效 | 技術 |
|---|---|---|
| 首頁 Hero | 文字逐字浮現 + 視差 | GSAP SplitText |
| 全站滾動 | 滑順滾動 | Lenis |
| 區塊出現 | 淡入上移 20px | Motion One |
| 頁面切換 | 平滑過渡 | Astro View Transitions |
| 師資卡片 | hover 3D 微傾斜（最大 5°） | Vanilla Tilt |
| CTA 按鈕 | 磁吸效果 | GSAP |
| 課程橫幅 | 跑馬燈科目名 | CSS animation |
| 數據區 | 數字滾動計數 | CountUp.js |
| 載入畫面 | 粉筆寫字 SVG | SMIL / GSAP DrawSVG |

## 7. 應用範例規範

### 7.1 網站元件

| 元件 | 規範 |
|---|---|
| **按鈕** | 主按鈕：Caramel Gold 背景 + Charcoal Bean 文字；圓角 8px；hover 略升 1px + 加深 |
| **卡片** | Chalk White 背景 + 1px Cream Beige 邊框；圓角 12px；hover 微浮起 + 陰影 |
| **連結** | 預設 Espresso Tan，hover 加底線（手繪粉筆線感） |
| **表單** | 輸入框圓角 6px，focus 時邊框轉 Caramel Gold |
| **Tag/Chip** | Cream Beige 背景 + Espresso Tan 文字，圓角 999px |

### 7.2 FB / IG 貼文圖卡規範

| 項目 | 規範 |
|---|---|
| **尺寸** | 方形 1080x1080；直式 1080x1350；限動 1080x1920 |
| **背景** | Latte Foam 或 Cream Beige，可加細微紋理 |
| **標題位置** | 上方或左上 1/3，留白充足 |
| **字級** | 標題不小於 56px（手機可讀） |
| **品牌標示** | 角落小字「賈伯斯數理教室 · 嘉義」，不超過 20px |
| **CTA** | 底部右下，焦糖金按鈕 |
| **避免** | 滿版背景圖、字蓋在複雜照片上、四個角全塞滿 |

### 7.3 短影音字卡規範

| 項目 | 規範 |
|---|---|
| **字幕字體** | 思源黑體 Bold |
| **字幕位置** | 下方 1/4，置中 |
| **字幕色** | 白色 + 黑色描邊 0.5px（保證可讀） |
| **重點字** | 用 Caramel Gold 高亮 |
| **片頭** | 1 秒 Logo 浮現（粉筆寫字感） |
| **片尾** | 2-3 秒 CTA：「加 LINE 領取資料」+ QR code |
| **配樂** | 平靜的 lo-fi、爵士、無人聲鋼琴；避免 EDM、流行歌 |
| **節奏** | 30 秒影片至少 4-6 個剪接點，但不要過度卡點 |

### 7.4 名片 / 文宣規範（印刷品）

- 紙質：建議馬糞紙、米色卡紙、巴川紙（呼應咖啡館感）
- 印刷：黑白為主，重點處燙金（呼應 Caramel Gold）
- 避免：亮面銅版紙、UV 上光、雷射全像

## 9. 品質審稿 Checklist（驗收必看）

每收到外包稿件，請逐項勾選：

### 視覺基礎
- [ ] 配色是否在指定 5-7 色內？沒有出現禁忌色？
- [ ] 字體是否只用了規範內的 2-3 種？
- [ ] Logo / 品牌標示是否正確使用（位置、尺寸、色彩）？

### 內容正確性
- [ ] 文字是否有錯別字？特別是學校名（嘉中 / 嘉女 / 興華 / 協同）
- [ ] 數據是否正確？（13 年資歷、台清交師資、嘉義東區地址）
- [ ] 課程資訊、價格、時段是否與最新版一致？

### 品牌調性
- [ ] 是否符合「溫潤、專業、不浮誇」的品牌個性？
- [ ] 是否避免了禁用詞彙（保證、絕對、最強等）？
- [ ] 整體感覺是否「像咖啡館，不像企業」？

### 行動裝置
- [ ] 字級在手機上是否可讀（標題 ≥ 56px）？
- [ ] 重要資訊是否在折疊區以上？
- [ ] CTA 按鈕是否好點擊（≥ 44x44px）？

### CTA 與轉換
- [ ] 是否有清楚的 CTA？（試聽？加 LINE？）
- [ ] CTA 文案是否具體（「立即試聽」優於「了解更多」）？

### 技術交付
- [ ] Source 檔是否齊全（AI / PSD / Figma / Premiere project）？
- [ ] 字體是否轉外框（避免對方沒裝字體）？
- [ ] 影片是否有 1080p 與 4K 版本？
