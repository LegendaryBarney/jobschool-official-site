## 為什麼學 Python

我們不主張「孩子一定要會寫程式」，但在 AI 變得日常的時代，**能用 Python 寫一段小工具的孩子，會比只能用工具的孩子，多一個和世界互動的方式**。Python 是入門最友善、實用性最高的語言，從資料處理、自動化到 AI，都能用 Python 開始。

## 教學節奏：依程度給速度與難度

- **基礎弱的同學**：從變數、條件、迴圈一步一步建立邏輯感
- **程度好的同學**：補充演算法、資料結構、AI / 資料分析的入門題型
- **程度中等的同學**：訓練「把問題拆成小步驟」的程式設計思維

## 教學重點

- **基礎語法**：變數、條件判斷、迴圈、函數
- **資料結構**：list、dict、set、tuple
- **檔案與字串處理**：CSV、JSON 入門
- **簡易專題**：自動化小工具、簡單爬蟲、資料視覺化
- **進階引導**：依學生興趣導向 AI 或演算法競賽方向

## 由臺大資工碩士親自授課

Barney 老師畢業於臺大資工碩士，本科就是寫程式。在我們的經驗裡，**教程式最大的差別不是「會不會寫」，而是「能不能把為什麼這樣寫講清楚」**。Barney 老師的板書與口語邏輯，讓他在數學課的優勢，同樣完整轉移到程式設計課堂上。

## 1 節免費試聽

本課程提供 **1 節免費試聽**。寫程式這件事，孩子需要親自體驗才能真正知道自己是否有興趣，所以我們提供免費試聽，讓家長與孩子先實際感受課堂節奏，再決定是否報名。
